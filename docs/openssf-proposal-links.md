## BadgeApp proposal notes

When updating the BadgeApp project, do not overclaim Gold. Use `Passed` only for
items with repository evidence or maintainer-confirmed settings. Use `Partial` or
`Needs human confirmation` for branch protection details, private vulnerability
reporting, secret scanning/push protection, and independent review until verified
in GitHub settings and recent PR history.
