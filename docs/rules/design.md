# Design Rules

- [design.copper-balance](design.copper-balance.md): Checks filled copper area per layer against board area to identify low copper coverage.
- [design.board-outline](design.board-outline.md): Checks that the PCB Edge.Cuts outline is present and closed.
- [design.unique-references](design.unique-references.md): Checks board footprints for duplicate reference designators.
