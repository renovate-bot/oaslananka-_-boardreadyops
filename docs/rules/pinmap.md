# Pinmap Rules

- [pinmap.verify](pinmap.verify.md): Checks configured pinmap nets against schematic net labels.
- [pinmap.unmapped-pin](pinmap.unmapped-pin.md): Checks connected schematic pins against pinmap entries.
- [pinmap.collision](pinmap.collision.md): Checks pinmap files for duplicate pin or net assignments.
- [pinmap.net-label](pinmap.net-label.md): Checks pinmap net names against schematic global, local, and hierarchical labels.
