# Release Rules

- [release.revision-set](release.revision-set.md): Checks board title-block revisions against the configured release tag pattern.
- [release.changelog-present](release.changelog-present.md): Checks CHANGELOG.md for an entry matching the current board revision.
- [release.tag-matches-revision](release.tag-matches-revision.md): Checks tag CI context against board revision.
- [release.version-format](release.version-format.md): Checks schematic and PCB revision strings against the configured release version pattern.
